What is the Document Dbject Model(DOM)

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects (nodes) that browsers use to display and interact with the page. Each part of an HTML or XML document—such as elements, attributes, and text content—is treated as a node in this tree, making it accessible and manipulatable using programming languages like JavaScript.

what is the event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you handle events at a parent element rather than assigning individual event listeners to multiple child elements. This approach leverages event bubbling, where an event triggered on a child element propagates up through its ancestors (parents), allowing the parent to detect and handle the event.

How do you manipulate an elements attribute and style using the DOM

In the DOM, you can manipulate an element's attributes and style using JavaScript. This allows you to dynamically change the appearance or behavior of elements on the page.
